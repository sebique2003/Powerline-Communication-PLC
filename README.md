## 📌 Project Overview
Technical analysis of **PLC** technology for high-speed data transmission over existing electrical infrastructure, developed for the *Local Networks* course at **IESC**.

## 🚀 Key Concepts
* **Superimposition:** Injecting data into the 50 Hz AC cycle.
* **Modulation:** Use of **OFDM** for noise resilience.
* **Standards:** IEEE 1901 & HomePlug AV2 protocols.

## 📊 Quick Comparison

| Feature | PLC | Ethernet | Wi-Fi 6 |
| :--- | :--- | :--- | :--- |
| **Install** | Plug & Play | High (Cables) | Moderate |
| **Cost** | Low | Moderate | Moderate |
| **Speed** | Up to 2 Gbps | 10 Gbps | 9.6 Gbps |

## 🛠 Applications
* **Smart Grids:** Monitoring UNITBV research labs.
* **IoT:** Home automation without new wiring.
* **Industrial:** LANs in high-interference zones.
